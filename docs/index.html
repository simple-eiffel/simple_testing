Enhanced Test Framework for Eiffel

Overview Classes Assertions Installation

Overview

simple_testing provides a rich set of assertion methods that extend the standard EQA_TEST_SET framework. Inherit from TEST_SET_BASE to get access to 40+ assertion methods for booleans, numbers, strings, collections, and object references.

Classes

TEST_SET_BASE

Deferred base class providing 40+ assertion methods. Inherit from this class instead of EQA_TEST_SET.

  • Boolean assertions
  • Object/Reference assertions
  • Numeric assertions (INTEGER, REAL_64, NATURAL_64)
  • String assertions with detailed diff
  • Collection assertions

TEST_SET_BRIDGE

Bridge class for accessing features exported to {NONE} in tests.

Export your internal features to {TEST_SET_BRIDGE} and inherit from this class in your test set to access them.

Installation

1. Set Environment Variable

SIMPLE_TESTING=D:\prod\simple_testing

2. Add to ECF

<library name="simple_testing" location="$SIMPLE_TESTING\simple_testing.ecf"/>
