Overview

simple_testing provides 40+ assertion methods that extend the standard EQA_TEST_SET framework. Inherit from TEST_SET_BASE to get rich assertions for booleans, numbers, strings, collections, and object references with detailed failure messages.

Features

Boolean Assertions

assert_true, assert_false, refute

Object Assertions

assert_attached, assert_void, assert_equal, assert_same_reference

Integer Assertions

assert_greater_than, assert_less_than, assert_in_range, assert_positive

Real Assertions

assert_reals_equal (with epsilon), assert_real_in_range

String Assertions

assert_string_contains, assert_starts_with, assert_strings_equal_diff

Collection Assertions

assert_array_has_item, assert_iterable_is_empty, assert_count_equals

Installation

<library name="simple_testing" location="$SIMPLE_EIFFEL/simple_testing/simple_testing.ecf"/>

Quick Start

class
    MY_TEST_SET

inherit
    TEST_SET_BASE

feature -- Tests

    test_user_creation
        local
            user: USER
        do
            create user.make ("Alice", 25)

            -- Boolean assertions
            assert_true ("is_active", user.is_active)

            -- String assertions
            assert_string_contains ("has_name", user.name, "Ali")
            assert_strings_equal_diff ("exact_name", "Alice", user.name)

            -- Integer assertions
            assert_integers_equal ("age", 25, user.age)
            assert_in_range ("valid_age", user.age, 0, 120)

            -- Object assertions
            assert_attached ("has_profile", user.profile)
        end

end

API Reference

Boolean Assertions

assert_true (tag: STRING; condition: BOOLEAN)
assert_false (tag: STRING; condition: BOOLEAN)
refute (tag: STRING; condition: BOOLEAN)

Object/Reference Assertions

assert_attached (tag: STRING; object: detachable ANY)
assert_void (tag: STRING; object: detachable ANY)
assert_same_reference (tag: STRING; expected, actual: ANY)
assert_equal (tag: STRING; expected, actual: ANY)
assert_not_equal (tag: STRING; expected, actual: ANY)

Integer Assertions

assert_integers_equal (tag: STRING; expected, actual: INTEGER)
assert_greater_than (tag: STRING; value, threshold: INTEGER)
assert_less_than (tag: STRING; value, threshold: INTEGER)
assert_in_range (tag: STRING; value, min, max: INTEGER)
assert_positive (tag: STRING; value: INTEGER)
assert_negative (tag: STRING; value: INTEGER)
assert_zero (tag: STRING; value: INTEGER)

Real Assertions

assert_reals_equal (tag: STRING; expected, actual, epsilon: REAL_64)
assert_real_in_range (tag: STRING; value, min, max: REAL_64)

String Assertions

assert_string_contains (tag, haystack, needle: STRING)
assert_string_starts_with (tag, str, prefix: STRING)
assert_string_ends_with (tag, str, suffix: STRING)
assert_string_empty (tag, str: STRING)
assert_strings_equal_diff (tag, expected, actual: STRING)

Collection Assertions

assert_array_has_item (tag: STRING; arr: ARRAY [ANY]; item: ANY)
assert_iterable_is_empty (tag: STRING; iterable: ITERABLE [ANY])
assert_count_equals (tag: STRING; collection: FINITE [ANY]; expected: INTEGER)
